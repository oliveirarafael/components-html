Buttons

Nesse guia ensinará como construir um botão utilizando as classes em CSS. Todas as propriedades Defaults do buttons foram resetadas

Estrutura Básica

A class .button configura as propriedades: cursor e font-weight

Tamanho

A class .button-[tamanho] define o tamanho do Button. Os button tem cincos tamanhos diferentes, são eles: .button-extra-large, .button-large, .button-medium, .button-small e .button-extra-small
